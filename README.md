# Интерактивная викторина (Quiz)

Викторина по основам программирования на HTML, CSS и JavaScript.

## О проекте

Это интерактивное приложение, которое проверяет знания по основам программирования (с уклоном в Python). Проект выполнен в рамках учебного задания.

## Функционал

- 10 вопросов с вариантами ответов
- Прогресс-бар, который обновляется с каждым вопросом
- При выборе ответа правильный вариант подсвечивается зелёным, неправильный — красным
- Подсчёт набранных баллов
- Кнопка «Результаты» для просмотра текущего счёта в любой момент
- После последнего вопроса — экран с итоговым результатом и кнопкой «Начать заново»

## Технологии

- HTML5
- CSS3
- JavaScript (ES6)

## Как запустить

1. Скачайте или клонируйте репозиторий:
git clone https://github.com/YuliaMarakshina/my-quiz.git
